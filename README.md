# golang-full-stack
Go app which aims to deliver data merging tool

Tech stack:
- Go
- Python (scripts)
- MongoDB

For future plans:
- Migration from server to AWS serverless services

Run commands:
- Go server:
  - cd data-merging\backend\cmd\app
  - go run main.go
Python commands:
  - cd data_input
  - python audio_recognition.py
  - python photo_recognition.py
